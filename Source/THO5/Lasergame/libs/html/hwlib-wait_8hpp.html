void hwlib::wait_ns ( int_fast32_t  n)

delay n nanoseconds

A call of this function will take (at least) n nanoseconds. The value of n must be in the range 0 to 999999999 (up to 1 second). For longer delays the function wait_ms() can be used.

Note that there is no guaranteed upper bound on the delay time.
